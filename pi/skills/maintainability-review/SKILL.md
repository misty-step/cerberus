# Maintainability Focus

Prioritize future-change risk:
- mixed concerns and high cognitive complexity
- brittle tests or missing tests for risky logic
- hidden side effects and implicit behavior
- unclear naming that obscures intent

Flag concrete maintenance burden introduced by this PR.
