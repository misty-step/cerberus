Backlog 013

Rust Local Review Replay

Port the local diff review entrypoint into Rust as a fixture-backed CLI path that constructs `ReviewRequest.v1` and runs the existing review core.

Date: 2026-06-18

Branch: shape/rust-review-engine-backlog

Stop: if local review requires semantic routing or live reviewer providers.

Target Outcome

`cerberus-cli review-local --diff-file ... --out ...` emits the same review artifact surfaces as `review --fixture`, using a `ReviewRequest.v1` built from a local git-style diff.

Design

Add a CLI-local parser for exact git diff syntax: `diff --git`, file status markers, and hunk +/- line counts. Then call the existing Rust `review` core and render JSON plus Markdown.

Acceptance Oracle

  • `cargo test --workspace local_review`
  • `cargo run --locked -p cerberus-cli -- review-local --diff-file fixtures/local-review/local.diff --out tmp/local-review`
  • `cargo run --locked -p cerberus-cli -- validate tmp/local-review/review-run-artifact.json`
  • Full Rust, shell, Node, and legacy Elixir gates remain green.

Request Shape

ReviewSource::LocalDiff { repo_path }
Change {
  title: "Local diff review",
  diff: raw_diff,
  files: parsed git diff files
}
ReviewContext {
  summary: "Rust local diff review."
}

Boundaries

  • No LLM router or semantic reviewer selection.
  • No live peer/provider calls.
  • No shelling out to `git` in this slice.
  • No replacement of the public action path.

Files

  • `crates/cerberus-cli/src/local_review.rs` for local diff parsing.
  • `crates/cerberus-cli/src/main.rs` for CLI command wiring.
  • `fixtures/local-review/local.diff` for replay proof.
  • Backlog and shaping docs for receipt.

Critic Focus

  • Parser fails closed on non-git diffs or empty file lists.
  • Added/copied/renamed/deleted statuses do not corrupt coverage.
  • The local path does not weaken core artifact validation.

Next Slice Boundary

After 013, route policy can be modeled as explicit config data or the hosted API request/response fixtures can be captured. This slice only proves local diff replay.
